
#Smart Building Energy Management System

## Overview
A machine learning project that predicts and optimizes building operations such as AC status, fan speed, lighting control, and temperature settings based on environmental and occupancy data.

## Technologies Used
* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Key Features
* Data Cleaning and Preprocessing
* Feature Engineering
* Random Forest Classification
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Linear Regression
* Feature Importance Analysis
* Energy Optimization Insights

## Future Improvements
* IoT Sensor Integration
* Real-Time Monitoring Dashboard
* Energy Consumption Forecasting
